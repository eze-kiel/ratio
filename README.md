# ratio

Generate differents ratio of an image.

## Installation

```
$ git clone https://github.com/eze-kiel/ratio.git
$ cd ratio
$ go build
```

## Usage

```
Usage: ratio [width] [height]
```

Example:

```
$ ratio 1920 1080
%	width	height
10	192	108
20	384	216
30	576	324
40	768	432
50	960	540
60	1152	648
70	1344	756
80	1536	864
90	1728	972
100	1920	1080
```
